🏁 What Is This?
We're students (ages 12–18) from Peninsula Baptist Church in Mooresville, NC competing in the AI Grand Prix — a global autonomous drone racing competition by Anduril Industries.
The task: Write Python code that navigates a drone through gates autonomously — gate recognition, path planning, and drone control. Fastest time wins. No pilots. $500,000 prize pool.
The path: Virtual Qualifier 1 (May) → Virtual Qualifier 2 (June) → Physical Qualifier in SoCal (Sept) → Finals in Ohio (Nov)
