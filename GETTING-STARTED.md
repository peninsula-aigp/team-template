# 🏁 Getting Started — Peninsula AIGP

**Hey! You're officially part of the AI Grand Prix team. Here's what to do this week.**

This should take about 30–45 minutes total. Do steps 1–4 tonight or tomorrow. The rest can wait until camp starts, but the sooner the better.

---

## Step 1: Create Your GitHub Account (5 min)

1. Go to **github.com** and click **Sign up**
2. Pick a professional username (your real name is fine — like `jemma-juras` or `jagger-j`)
3. Use your personal email
4. Verify your email when GitHub sends the confirmation
5. Add a profile photo and a short bio: *"AI Grand Prix competitor | Peninsula AIGP"*

**When done:** Send Dad your GitHub username so he can add you to the team.

---

## Step 2: Apply for the GitHub Student Developer Pack (5 min)

1. Go to **education.github.com/pack**
2. Click **Sign up for Student Developer Pack**
3. Select **Student**
4. Upload proof: school ID, report card, enrollment letter, or transcript
5. Add your school name and expected graduation year
6. Submit — approval usually takes 1–3 days

**Why this matters:** You get GitHub Copilot (AI that writes code with you), free cloud tools, and pro features. All free.

---

## Step 3: Create Your Claude Account (5 min)

1. Go to **claude.ai**
2. Click **Sign up** and create an account with your email
3. Dad may need to help with the account creation (18+ requirement)
4. Once you're in, start a new conversation and paste this **Master Prompt:**

```
Hi Claude! I'm [YOUR NAME], and I'm [YOUR AGE] years old. I'm a student at
Peninsula Baptist Church's AI Grand Prix Summer Camp in Mooresville, NC.

I'm a complete beginner. I've never coded before. Over the summer I need to learn
Python programming, math, AI fundamentals, and enough about autonomous systems
to compete in the AI Grand Prix — a global drone racing competition with a
$500K prize pool.

Your job as my AI tutor:
1. Guide me step by step. Never just give me answers.
2. Show the first step, let me try, then help if I'm stuck.
3. When I write code, review it honestly.
4. For math, teach the concept, do one example, then make me practice.
5. Connect everything to the AI Grand Prix.

Our guiding verse is Colossians 3:23: "Whatever you do, work at it with all
your heart, as working for the Lord."

Let's start! Help me understand what the AI Grand Prix is and what I need to learn.
```

5. Send it and see what Claude says!

---

## Step 4: Install Your Tools (15–20 min)

Install these in order:

| # | What | Where | Notes |
|---|------|-------|-------|
| 1 | **Python 3.12+** | python.org → Downloads | ⚠️ CHECK "Add Python to PATH" during install! |
| 2 | **VS Code** | code.visualstudio.com | Free code editor |
| 3 | **Python Extension** | Inside VS Code → Extensions (Ctrl+Shift+X) → search "Python" | Install the one by Microsoft |
| 4 | **GitHub Copilot** | Inside VS Code → Extensions → search "GitHub Copilot" | Sign in with your GitHub Student account |
| 5 | **Git** | git-scm.com | Use all default settings during install |
| 6 | **GitHub Desktop** | desktop.github.com | Easier way to push code — install this |

---

## Step 5: Check Your Computer (2 min)

The AI Grand Prix simulator requires a **Windows PC with a dedicated graphics card**. Check yours:

- [ ] Windows operating system (not Chromebook, not Mac)
- [ ] Dedicated GPU: NVIDIA RTX 2060 Super or better (or AMD equivalent)
- [ ] 16 GB RAM
- [ ] 60 GB free storage

**How to check your GPU:** Press `Win + R`, type `dxdiag`, press Enter, click the "Display" tab. It shows your graphics card.

**If you're not sure your computer works:** Tell Dad. We'll figure it out before camp starts.

---

## Step 6: Accept the Team Invite (once Dad adds you)

1. Dad will invite you to the **peninsula-aigp** GitHub organization
2. Check your email for the invite, or go to **github.com/peninsula-aigp** and look for a banner
3. Click **Accept**
4. Go to **github.com/peninsula-aigp/team-template/issues** and find your Setup checklist issue
5. Start checking off boxes as you complete each step!
