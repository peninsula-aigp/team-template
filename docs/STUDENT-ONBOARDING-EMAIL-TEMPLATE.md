Welcome to the AI Grand Prix Summer Camp at Peninsula Baptist Church! You're officially part of something big — a global autonomous drone racing competition with a $500,000 prize pool. This is real.

Our GitHub organization is live, your setup checklist is ready, and the competition kicks off this summer. Here's what I need from you before camp starts. It takes about 30 minutes.

**STEP 1: Create your GitHub account (5 min)**
— Go to github.com and click Sign up
— Pick a professional username (like firstname-lastname)
— Use your personal email
— Verify your email
— Add a profile photo and bio: "AI Grand Prix competitor | Peninsula AIGP"
— TEXT/EMAIL ME YOUR USERNAME so I can add you to the team

**STEP 2: Apply for GitHub Student Developer Pack (5 min)**
— Go to education.github.com/pack
— Click Sign up for Student Developer Pack
— Select Student
— Upload your school ID or report card as proof
— This gives you free access to GitHub Copilot (AI that helps you code) and other pro tools

**STEP 3: Create your Claude AI account (5 min)**
— Go to claude.ai
— Sign up with your email (a parent may need to help with the 18+ requirement)
— Once you're in, start a conversation and just say hi. Play around with it.
— I'll give you a special prompt to paste in on the first day of camp

**STEP 4: Install these tools on your computer (15 min)**
1. Python — python.org → Downloads → Install. IMPORTANT: check the box that says "Add Python to PATH"
2. VS Code — code.visualstudio.com → Download → Install
3. GitHub Desktop — desktop.github.com → Download → Install

**STEP 5: Check your computer**
The competition simulator requires Windows + a dedicated graphics card. To check yours:
— Press Win + R
— Type dxdiag
— Press Enter
— Click the Display tab
— Send me what it says under "Name" (that's your graphics card)

If you have a Chromebook or a Mac, let me know now. We need to figure that out before camp starts.
